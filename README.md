# Systems Solver

Systems solver is a python program desinged to solve systems of equations with any number of variables.
~~~
2x^2 = 8

> x = 2
~~~
~~~
2x^3 + 3y + 2z = 25
5y - 2z + 3x^3 = 23
5z - x^3 + 7y = 14

> x = 2
> y = 1
> z = 3
~~~
~~~
2x^2 + 5y = 23
9y - 4x^2 = -27

> x = 3
> y = 1
~~~

Input is given through the equations.txt file; equations are separated through new lines.
